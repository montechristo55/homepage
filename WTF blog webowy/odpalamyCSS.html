Czym jest CSS?

CSS to skrót od Cascading Style Sheet co znaczy kaskadowe style arkuszy. Jest to jezyk służący do opisu i formy prezentacji (wyświetlania) treści dokumentów (stron internetowych) w przeglądarce:
wygląd-kolory-marginesy-tło/tła-kroje czcionki-itd…
Jak działa CSS w praktyce? CSS jest to kod wewnątrz pliku, który składa się z sekcji: selector { property:value; property:value; property:value; } *selector – wybrany element strony, którego cechy opisujemy pomiędzy klamrami;
Przykładowa stylizacja nagłówka mogłaby wyglądać następująco: h1 { h1 – nagłówek; color: blue; color: kolor elementów wew; font-size: 36px; rozmiar czcionki; font-weight: bold; grubośc czcionki; text-align: center; umiejscowienie tekstu; text-transform: uppercase; zamiana liter na wielkie; letter-spacing: 5px; rozsztrzelenie tekstu; }
