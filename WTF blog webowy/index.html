HTML 1.01

Czym jest HTML?

HTML to skrót od Hypertext Markup Language. Jest to nic innego jak jezyk opisowy oparty na tzw. znacznikach (tagach) opisujacy zawartość strony internetowej.
SEMANTYCZNY HTML oznacza precyzyjne definiowanie przeznaczenia poszczególnych części kodu źródłowego.
*semantyczny (HTML) – mający znaczenie i stąd opierający sie na znacznikach;

Znacznik (tag) to element składowy dokumentu HTML, który jest zapisywany z użyciem nawiasów ostrych;
I tak wyróżniamy:
- znacznik pojedynczy < img />
- znacznik podwójny < a> < /a>

Atrybut to cecha (właściwość) znacznika, zdefiniowana w jego wnętrzu, w postaci: nazwa_atrybutu=”wartość”

O znacznikach raz jeszcze. 
<znacznik atrybut=”wartość”>treść </znacznik>
Znacznik definiujący hiperłącze (link) to < a> < /a> natomiast href to adres strony (od hyper text reference), do której ma prowadzić link; i tak: <a href=”adres-strony.pl”>LINK </a> A wiec przykładowo jeżeli chcemy umieścić link do strony .pl to zapiszemy:
< a href=”http://onet.pl” target=”_blank” title=”hint” >KLIKNIJ </a>
*na ekranie widać odnośnik KLIKNIJ, który jest linkiem do strony onet.pl;
Koniecznie trzeba pamiętać o umieszczeniu pełnego adresu, włącznie z protokołem http:// chyba, że adres mieści się w Twojej własnej domenie; target=_blank spowoduje, iż łącze otworzy się na nowej stronie; natomiast znacznik title=”hint” utworzy napis przy kursorze po najechaniu na link, tzw. hint;
Reasumując...HTML to język opisany za pomocą znaczników. Wyróżniamy znaczniki pojedyncze i podwójne. Znacznikom przypisywane są atrybuty, a tym konkretne wartości.

Czym jest CSS?

CSS to skrót od Cascading Style Sheet co znaczy kaskadowe style arkuszy. CSS to jezyk służący do opisu formy prezentacji (wyświetlania) treści dokumentów (stron internetowych) w przeglądarce; wygląd-kolory-marginesy-tło/tła-kroje czcionki-itd…
Jak działa CSS w praktyce? CSS jest to kod wewnątrz pliku, który składa się z sekcji: selector { property:value; property:value; property:value; } *selector – wybrany element strony, którego cechy opisujemy pomiędzy klamrami; Przykładowa stylizacja nagłówka mogłaby wyglądać następująco: h1 { h1 – nagłówek; color: blue; color: kolor elementów wew; font-size: 36px; rozmiar czcionki; font-weight: bold; grubośc czcionki; text-align: center; umiejscowienie tekstu; text-transform: uppercase; zamiana liter na wielkie; letter-spacing: 5px; rozsztrzelenie tekstu; }
